# Ruakh

*Ruakh* is a non-commercial website project that allows users to access to free resources
about the bible.

## Features

* Videos
* Bible verses
* Worship
* Bible studies
* Forum / Discuss
* Timer
* Live

## Goal of the website

### Use technologies :
* PHP 8
* Docker
* Jenkins (CI)
* PHPStan (static analyzer)
* ReactJS
* Symfony
* APIPlatform
* Git
* Testing
